# Public Showcase Boundary

This repository is a public showcase of NONO-GATE behavior and positioning.

## What is included
- Public documentation (README + briefs)
- A reproducible demo structure (PASS / FAIL scenarios)
- Example policy file for evaluation

## What is NOT included
- Core engine source code
- Any private signing keys
- Any internal governance keys
- Any proprietary release tooling

## Why
This repo is intended for evaluation and understanding,
not for copying the engine implementation.

If you need the engine for enterprise evaluation,
request the signed evaluation bundle.
